MySQL Function Groups

Function Group Description Examples
String Functions Functions for text manipulation. REPLACE(), SUBSTRING(), CONCAT(), LENGTH()
Type Conversion Functions Functions to convert between different data types. CAST(), CONVERT()
Aggregate Functions Functions that operate on multiple rows and return a single result. AVG(), SUM(), COUNT(), MAX(), MIN()
Mathematical Functions Functions that perform mathematical operations. ABS(), CEIL(), FLOOR(), ROUND()
Date and Time Functions Functions for handling date and time values. NOW(), CURDATE(), DATE_FORMAT()
Control Flow Functions Functions that control logic in SQL expressions. IF(), CASE, IFNULL(), COALESCE()
